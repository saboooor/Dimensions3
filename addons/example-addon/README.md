# Example Addon

A demonstration addon for the **Dimensions** plugin.

## Features
- Listens to the `CustomPortalIgniteEvent`.
- Checks if the igniting entity is a player and has the permission `dimensions.exampleaddon.explosion`.
- Spawns simple explosion particles at the center of the portal upon successful ignition.

This addon serves as a minimal template/starting point for developers wishing to write custom addons using the Dimensions API.
