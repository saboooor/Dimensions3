# Portal TP Addon

A utility addon for the **Dimensions** plugin that adds a command to **teleport a player to the nearest active portal** of a given type.

## Features
- Provides a `/dim tpToClosest <portalName> [player]` command.
- Finds the nearest ignited portal of the specified type and teleports the target player to it.
- Useful for server operators or adventure maps that need to programmatically move players to portals.

## Commands
| Command | Description |
|---|---|
| `/dim tpToClosest <portalName> [player]` | Teleport a player to the nearest portal of the given type |
