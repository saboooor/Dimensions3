# WorldGuard Flags Addon

A compatibility addon for the **Dimensions** plugin that integrates custom flags into **WorldGuard** to control portal interactions within regions.

## Custom Flags
- `ignite-custom-portal` (StateFlag) - Controls whether players can ignite custom portals in a region.
- `destroy-custom-portal` (StateFlag) - Controls whether players can destroy/break custom portals in a region.
- `use-custom-portal` (StateFlag) - Controls whether players can use custom portals to teleport in a region.
